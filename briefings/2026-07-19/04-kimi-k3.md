# 오늘의 AI 개념: 키미 K3(Kimi K3) 출시

> 작성일: 2026-07-19 · 분류: trend

## 한 줄 정의

중국 문상AI(Moonshot AI)가 공개한, 지금까지 나온 오픈웨이트(가중치 공개) AI 모델 중 가장 큰 규모의 모델이다.

## 쉬운 설명

키미 K3는 2.8조 개의 파라미터(매개변수)를 가진 초대형 언어모델로, 가중치가 공개된 모델 중 역대 최대 규모다. 여기서 오픈웨이트(open-weight)란 모델을 학습시킨 내부 수치(가중치)를 누구나 내려받아 자체 서버에서 돌릴 수 있게 공개한다는 뜻으로, 소스코드까지 전부 공개하는 완전한 오픈소스와는 다르지만 실질적으로 자유롭게 활용할 수 있다.

비유하자면 초대형 백화점 하나를 통째로 짓는 대신, 2.8조 개의 매장 중 질문마다 필요한 매장 16곳만 열어 손님을 받는 방식이다. 실제로 키미 K3는 전문가 혼합(Mixture-of-Experts, MoE) 구조를 써서 896개의 전문가(부분 신경망) 중 토큰마다 16개, 즉 전체의 약 1.8%만 활성화한다. 이 덕분에 모델 크기는 압도적으로 크지만 실제 연산량은 그보다 훨씬 작게 유지된다.

기존에 다룬 소형언어모델(SLM)이 "작게 만들어 효율을 얻는" 접근이라면, 키미 K3는 "크게 만들되 일부만 골라 쓰는" 접근이라는 점에서 방향이 반대다. 둘 다 같은 목표, 즉 성능 대비 연산 비용을 낮추는 데 도달하려 한다는 공통점이 있다.

## 동작 원리

키미 K3의 핵심 기술 요소는 다음과 같다.

1. **전문가 혼합(MoE) 구조**: 전체 896개 전문가 중 입력 토큰마다 16개만 활성화해 연산량을 줄인다.
2. **키미 델타 어텐션(Kimi Delta Attention)**: 선형 어텐션 방식을 결합한 하이브리드 구조로, 긴 문맥을 더 적은 연산으로 처리한다.
3. **어텐션 잔차(Attention Residuals)**: 레이어 간 정보 전달 방식을 바꿔 학습 효율을 높인다.
4. 이런 구조 변경을 통해 이전 모델 키미 K2 대비 약 2.5배의 스케일링 효율 개선을 이뤘다고 문상AI는 밝혔다.

결과적으로 100만 토큰 컨텍스트 창과 네이티브 비전(이미지 이해) 기능을 지원하면서도, 실제 서비스 비용은 초거대 모델치고 낮은 수준으로 책정할 수 있게 됐다.

## 비슷한 것과 비교

| 구분 | 키미 K3 | 클로드 페이블 5 / GPT-5.6 솔 | 소형언어모델(SLM) |
|------|---------|------------------------------|---------------------|
| 파라미터 규모 | 2.8조(오픈웨이트 최대) | 비공개(폐쇄형 최상위권) | 수십억~수백억 |
| 공개 방식 | 가중치 공개(오픈웨이트) | 비공개(API로만 제공) | 오픈웨이트인 경우 많음 |
| 종합 성능 | 최상위권에 근접, 일부 벤치마크 1위 | 종합 성능 최상위 | 특정 업무에 특화, 종합 성능은 낮음 |
| 적합한 상황 | 자체 서버 운영, 비용 대비 성능 중시 | 최고 품질이 필요할 때 | 단순 반복 작업, 온디바이스 실행 |

성능 최상위가 필요하면 폐쇄형 최상위 모델을, 자체 인프라에 설치해 데이터를 외부로 내보내지 않아야 하면 키미 K3 같은 오픈웨이트 모델을, 단순 반복 작업이나 기기 내 실행이 목적이면 SLM을 고르는 것이 기본 선택 기준이다.

## 핵심 용어·논쟁

- **오픈웨이트(Open-weight)** — 모델 학습 결과인 가중치 파일을 공개해 누구나 내려받아 실행할 수 있게 한 배포 방식. 학습 코드·데이터까지 공개하는 완전한 오픈소스와는 구분된다.
- **전문가 혼합(Mixture-of-Experts, MoE)** — 여러 개의 작은 신경망(전문가) 중 입력마다 일부만 선택적으로 활성화해 연산량을 줄이는 모델 구조.
- **파라미터(Parameter, 매개변수)** — 모델이 학습을 통해 값을 조정하는 내부 수치. 흔히 모델 규모를 나타내는 지표로 쓰인다.
- **스케일링 효율(Scaling Efficiency)** — 모델 규모를 키울 때 성능이 얼마나 비례해서 향상되는지를 나타내는 척도.

중국계 기업이 미국의 반도체 수출 규제 속에서도 최상위권에 근접한 모델을 내놓은 것을 두고, 규제의 실효성과 컴퓨팅 자원 확보 우회 경로에 대한 논쟁이 이어지고 있다. 이는 아직 결론이 나지 않은 진행형 쟁점이다.

## 왜 지금 중요한가

문상AI는 2026년 7월 중순 키미 K3를 공개했으며, 코딩·에이전트 벤치마크에서 클로드 오퍼스 4.8과 GPT-5.5를 넘어섰다고 밝혔다. 다만 종합 성능에서는 앤트로픽의 클로드 페이블 5와 오픈AI의 GPT-5.6 솔(Sol)에는 아직 못 미친다고 자체 평가했다. 전체 가중치는 2026년 7월 27일까지 수정된 MIT 유사 라이선스로 공개될 예정이다.

이 발표는 미국의 대중국 반도체 수출 규제 속에서도 중국 기업이 최상위권에 근접한 오픈웨이트 모델을 내놓았다는 점에서 업계에 충격을 줬고, 반도체 관련 주가가 하락하는 등 시장 반응으로도 이어졌다. API 가격은 캐시 적중 시 입력 토큰 100만 개당 0.30달러, 캐시 미스 시 3달러, 출력 토큰 100만 개당 15달러로 책정됐다.

## 회계법인 AI 직무 연결 포인트

회계법인이 AI 도구를 도입할 때는 특정 모델 하나에 종속되지 않고 여러 모델을 검토하는 벤더 다변화 전략이 중요한데, 키미 K3처럼 오픈웨이트로 공개되는 초대형 모델은 자체 서버에 직접 설치해 운영할 수 있어 데이터를 외부로 내보내지 않는 온프레미스(내부망) 감사 데이터 분석 환경을 구성하는 선택지가 될 수 있다.

다만 중국계 모델을 감사·재무 데이터 처리에 쓰려면 국내 개인정보보호법·AI기본법상 데이터 처리 위치, 보안 인증, 규제 준수 여부를 별도로 검토해야 하므로, 성능만으로 도입을 결정할 수 없다는 점도 실무에서는 함께 고려해야 한다.
