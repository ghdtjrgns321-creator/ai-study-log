# AI 학습 로그

회계·감사 실무자를 위한 **AI 개념 매일 학습 저장소**. 매일 아침 7시(KST), 클라우드 예약 에이전트가
그날 좋은 주제 3~5개를 검색·선별해 브리핑을 작성하고 자동 커밋한다. 읽는 사람은 아침에 열어 공부만 하면 된다.

## 구조

```
ai-study-log/
├─ README.md                  이 문서
├─ covered.md                 다룬 주제 대장(중복 방지 겸 학습 이력 인덱스)
├─ agent/
│  ├─ instructions.md         에이전트 지시서(선정 규칙·형식·커밋)
│  └─ template.md             브리핑 출력 형식
└─ briefings/
   ├─ 2026-07-06.md           (초기 수동 작성분)
   └─ YYYY-MM-DD/NN-슬러그.md  (자동 생성분, 날짜 폴더)
```

## 매일 어떻게 채워지나

1. 에이전트가 저장소를 클론한다.
2. `agent/instructions.md`를 읽고 그대로 수행한다 — `covered.md`로 중복을 피하고, 웹 검색으로 후보를
   모아 품질순 3~5개를 고른 뒤 `agent/template.md` 형식으로 작성한다.
3. `covered.md`를 갱신하고 커밋·푸시한다.

## 방식을 바꾸고 싶다면

에이전트 행동은 전부 `agent/instructions.md`에 있다. 주제 분류, 선정 기준, 신선도 규칙, 하루 편수 등을
이 파일에서 고치면 다음 실행부터 반영된다. 코드나 예약 설정을 건드릴 필요 없다.
